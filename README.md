Para ejecutar este proyecto necesitas tener instalado en tu computadora:

- **Node.js** (Versión 16 o superior recomendada).
- **NPM** (Viene instalado con Node.js).

Puedes verificar si los tienes instalados ejecutando en tu terminal:

```bash
node -v
npm -v
```

Para hacer correr este proyecto deberás usar el comando:

```bash
npm run dev
```
